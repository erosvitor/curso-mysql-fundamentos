## Sobre
O projeto é um curso sobre os fundamentos do banco de dados MySQL.

## Tópicos disponíveis
* Introdução ao curso
* Noções sobre banco de dados
* Instalando o MySQL
* Acessando o MySQL
* Criando um banco de dados
* Alterando tabelas
* Manipulando dados
* Filtrando e agrupando dados
* Relacionamento 1:1
* Relacionamento 1:N
* Relacionamento N:N
* MySQL Workbench
* Stored procedures
* Transações
* Subselects
* Functions
* Triggers
* Segurança
* Backup e Restore
