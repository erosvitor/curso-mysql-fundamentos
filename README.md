## Sobre
Teoria, exemplos e exercícios sobre os fundamentos do banco de dados relacional MySQL.

## Tópicos
* Introdução a banco de dados
* Criando um banco de dados
* Alterando tabelas
* Manipulando dados
* Filtrando e agrupando dados
* Relacionamento 1:1
* Relacionamento 1:N
* Relacionamento N:N
* MySQL Workbench
* Stored procedures
* Transações
* Subselects
* Functions
* Triggers
* Segurança
* Backup e Restore
